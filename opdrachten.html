Project CV

Beschrijving:

Ik moest een response CV maken in websitevorm. Dit is een groot project waar ik gedurende het semester veel aan gewerkt heb.

Persoonlijke uitdagingen:

De grootste persoonlijke uitdagingen waren de layout en het javascript. Layout in CSS vind ik moeilijk en het is soms echt even spelen met dingen om ze op de juiste plaats te krijgen. Ik kende ook nog niks van javascript maar toch wou ik er iets mee doen. Ik ben er na veel tijd en moeite in geslaagd om een simpel script te schrijven. Dit script laat via een button click een tekst verschijnen in typing animatie. Het was ook een uitdaging om mijn site responsive te maken.

Wat ik heb bijgeleerd:

Ik heb iets bijgeleerd over javascript. Ik heb ook veel bijgeleerd over flexboxes om mijn containers te stylen. Responsive webdesign is ook iets dat ik heb bijgeleerd.
